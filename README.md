IOS 14 ve üzeri yazılımların desteklediği bir özwlliğide ekledim page control üzerinde parmağınızı 
kaldırmadan gezebiliyorsunuz. Trendyol gibi ürün görseli paylaşan şirketler kullanıyorlar applerinde.
